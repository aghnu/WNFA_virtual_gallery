# WNFA Virtual Gallery
A 3D Virtual Gallery displays the poster results generated by WNFA at Zhejiang Exhibition Hall from 06.01.2022 to 06.12.2022. It is written in vanilla JavaScript. 3D is achieved by CSS 3D transform. https://www.aghnu.me/gallery/WNFA/

Here are some links:
- [The 3D virtual gallery for WNFA (Website Link)](https://www.aghnu.me/gallery/WNFA)
- [WNFA Online Poster Generator](https://www.aghnu.me/WNFA)
- [Original WNFA project (Django)](https://github.com/aghnu/WNFA)
